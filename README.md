# ClearImpact – Transparent Giving for Real Change

ClearImpact is a full-stack platform prioritizing transparency in donations. It allows donors to trace their exact funds, provides real-time impact generation capabilities, and creates a clear pathway to real-world change.

## Stack
- **Frontend**: React (Vite, TS), Tailwind, React Router
- **Backend**: Node.js, Express, TS, MongoDB

## Local Development

### 1. MongoDB Setup
Ensure MongoDB is running locally (`mongodb://localhost:27017`) or configure the `MONGODB_URI` properly.

### 2. Backend Setup
```bash
cd backend
npm install
# Create a .env file based on .env.example
npm run dev
```

### 3. Frontend Setup
```bash
cd frontend
npm install
# Create a .env file based on .env.example
npm run dev
```
